- Parameter `penalty` of :class:`linear_model.LogisticRegression` and
  :class:`linear_model.LogisticRegressionCV` is deprecated and will be removed in
  version 1.10.
  Use `l1_ratio=0` (or `l1_ratios=(0,)`) instead of `penalty="l2"`, `l1_ratio=1`
  instead of `penalty="l1"` and any other value between 0 and 1 for `l1_ratio` instead
  of `penalty="elasticnet"`; set `C=np.inf` instead of `penalty=None`.
  Additionally, the default value of `l1_ratio` of
  :class:`linear_model.LogisticRegression` changes from `None` to `0.0`.
  The default value  None of `l1_ratios` of :class:`linear_model.LogisticRegressionCV`
  is deprecated in will change from `None` to `(0.0,)` in version 1.10.
  This deprecation warning can be silenced by explicitly setting `l1_ratios=(0,)`.
  Setting either `l1_ratio` or `l1_ratios` to `None` is also deprecated and will raise an
  error in version 1.10.
  By :user:`Christian Lorentzen <lorentzenchr>`.
