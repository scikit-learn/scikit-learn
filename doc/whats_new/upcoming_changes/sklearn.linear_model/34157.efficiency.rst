- The `"newton-cholesky"` solver of :class:`linear_model.LogisticRegression`,
  :class:`linear_model.GammaRegressor`, :class:`linear_model.PoissonRegressor` and
  :class:`linear_model.TweedieRegressor` now uses an improved backtracking line search.
  Instead of halving the step size in each line search iteration, it uses cubic or
  quadratic polynomial interpolation to find the next trial step size.
  By :user:`Christian Lorentzen <lorentzenchr>`.
