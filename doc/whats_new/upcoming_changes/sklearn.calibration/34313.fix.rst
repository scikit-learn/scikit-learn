- Fixed :class:`calibration.CalibratedClassifierCV` with ``method="sigmoid"`` or
  ``method="isotonic"`` to convert :term:`predict_proba` outputs to logits
  before calibration instead of passing probabilities directly to the
  calibrator. This aligns Platt scaling with the intended logit input and
  improves calibration for estimators such as :class:`naive_bayes.GaussianNB`.
  For ``method="temperature"``, :term:`predict_proba` outputs are converted to
  multinomial logits. Estimators that only expose :term:`decision_function`
  are unaffected.
  By :user:`Olivier Grisel <ogrisel>` and :user:`Antoine Baker <antoinebaker>`.
